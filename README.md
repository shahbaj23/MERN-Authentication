A full-stack MERN (MongoDB, Express, React, Node.js) authentication system with JWT-based login and registration. This project demonstrates secure user authentication, including password hashing, validation, and session management.

Features

User registration with validation (email uniqueness, password length, password match)

User login with JWT token generation

Password hashing with bcrypt for security

Protected routes for authenticated users

Client-side form validation with React

Error and success notifications using react-toastify

Fully responsive and clean UI

Technologies Used

Frontend: React, Tailwind CSS, React Router, react-toastify

Backend: Node.js, Express.js, MongoDB, Mongoose, bcrypt, jsonwebtoken

Tools: Axios for API calls, Nodemon for development
